# Pong Clone
A simple Pong-inspired game made with Godot.

## About
This project recreates the classic Pong game with basic controls and gameplay.

## How to Play
- Use **W** and **S** or **arrow keys** to move the paddle.
- The goal is to score 10 points by making the ball pass the CPU's paddle.
